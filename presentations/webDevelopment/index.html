Basic Structure

HTML

  • Hypertext Markup Language
  • Proposal of 18 elements by Tim Berners-Lee in 1993
  • Influenced by Cern's SGML
  • IETF creates new standard in '94
  • W3C create new standard in '96
  • Became an ISO/IEC standard in 2000
  • WHATWG pushed for a new standard in '14

CSS

  • Born in 1996 in conjunction with the new HTML standard
  • The goal was to provide separation of content and presentation
  • This provided Accessability:
    • Screen readers
    • Captions/Titles/Sub-titles
    • Logical tab advancement
    • Simple and logical

JavaScript

  • Abbreviated as JS
  • High-level, interpreted programming language
  • Dynamic, weakly typed, prototype-based, and multi-paradigm
  • Developed as Mocha
  • Released as LiveScript in NetScape Navigator 2.0 Beta
  • Released as JavaScript in the official release

Future

  • Annual releases of ECMAScript
  • Web Assembly
